安装Docker:
1.sudo yum install -y yum-utils device-mapper-persistent-data lvm2
2.sudo yum-config-manager --enable extras
3.sudo yum-config-manager \ --add-repo \ https://download.docker.com/linux/centos/docker-ce.repo
4.sudo yum install docker

//安装镜像 设置端口 
docker run -p 5050 --name nginx -it ubuntu /bin/bash
docker run -p 5001 --name webapitwo -it centos /bin/bash

启动docker
sudo systemctl daemon-reload
sudo systemctl restart docker

添加开机自启
systemctl enable docker

查看安装效果
sudo docker run microsoft/dotnet-samples

查看Docker 版本
docker version

搜索可用的镜像版本
docker search tutorial

常用命令：
docker ps：列出所有正在运行的容器
docker image ls：列出所有的镜像
docker stop <name>：暂停容器do
docker rm <name>：删除容器

要删除全部image的话
docker rmi $(docker images -q)
